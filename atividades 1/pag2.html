PLANO DE ENSINO

Justificativa

A área de programação para ambientes web é hoje elemento essencial na formação profissional do aluno do curso de Sistemas de Informação. Esta importância existe em virtude da presença massiva da Internet nos diversos setores da sociedade. Neste sentido, consonante com a proposta do curso de formar profissionais capacitados para desenvolver para e atuar em sistemas para Internet, o curso tem duas perspectivas fundamentais: a de projeto e desenvolvimento de sistemas para Internet. No primeiro busca-se trabalhar com as competências relacionadas ao projeto de ambientes web, e em especial aspectos como usabilidade e acessibilidade na web; no segundo, busca-se apresentar e trabalhar com as diversas ferramentas, métodos e tecnologias para o desenvolvimento de ambientes web. Tais conteúdos tem enfoque no chamado front-end, ou seja, nos aspectos visuais e de interação presentes no lado do usuário, haja visto o enfoque do servidor ser objeto de disciplina posterior. Espera-se assim que o aluno, na conclusão da disciplina, possua conhecimento necessário para o projeto e desenvolvimento do front-end de ambientes web, seja os projetando ou então atuando como gestores e empreendedores, como preza o Projeto Pedagógico deste curso.

Objetivo Geral

O objetivo geral desta disciplina é propiciar aos alunos condições para projetar e desenvolver a parte de front-end de programas executados na web, com as tecnologias e técnicas adequadas a este fim.

Objetivos Específicos

  1. Compreender as tecnologias de desenvolvimento de front-end na web (HTML, CSS e Javascript).
  2. Compreender coneitos básicos para a construção de layouts para a web.
  3. Compreender conceitos de usabildiade e acessiblidade para a web.
